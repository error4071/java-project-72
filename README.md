## Анализатор страниц.

Данный проект позволяет анализировать указанные страницы на SEO пригодность.

### Использовавшийся стек инструментов:

+ Java 20;
+ Gradle 8.3;
+ Javalin
+ H2, PostgreSQL;
+ Lombok;
+ JUnit

## Требования: 

### Для запуска приложения необходимо установить следующие зависимости:

+ JDK 20
+ Gradle 8.3
+ Node.js 16.13.1
+ Makefile
